# ShopifyBuy.UI [WIP]

## Development

```
npm install
npm run start

```

Will watch for changes, compile src/ to build/ using babel & browserify, and run a server on port 8080.

You can acess running examples at:
- http://127.0.0.1:8080/examples/

## Testing

```
npm run test:dev
```

Will watch for changes and run test suite.
